Política de Privacidade

Última atualização: 23 de Abril de 2026

1. Introdução

Esta Política de Privacidade descreve como a Central de Gestão (Escola de Inglês) coleta, usa e protege suas informações ao utilizar nosso CRM PWA.

2. Coleta de Dados

Coletamos informações necessárias para a gestão acadêmica e comercial, incluindo:

  • Dados de contato (nome, e-mail, telefone).
  • Informações de alunos e progresso pedagógico.
  • Dados financeiros para controle de cobranças.
  • Acesso ao Google Calendar (quando autorizado) para sincronização de aulas.

3. Uso do Google Calendar API

Nosso aplicativo solicita permissão para acessar sua agenda do Google para:

  • Criar eventos de aulas automaticamente.
  • Listar seus compromissos para evitar conflitos de horários.

Nós não compartilhamos nem vendemos seus dados do Google para terceiros.

4. Armazenamento de Dados

Todos os dados são armazenados de forma segura utilizando o provedor Supabase, com criptografia e autenticação de ponta.
